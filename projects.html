Projects

I have done several projects in past four years at IIT Bombay. Each project trying to explore a new dimension in computer science through my courses

Texture Classification

Guide : Prof Suyash Awate - Autumn 2014

Designed and implemented image categorization and segmentation based on visual textures through a texton-based learning approach to model and represent visual textures. Supervised Learning performed using SVM and Decision Tress to create model for the database. Report

Virtual Memory for Experimental OS

Guide : Prof Dhananjay D Dhamdhere - Spring 2014

Designed and implemented effective data structures and algorithms for handling process memory allocation, swap space management, with process swap in and out on Input Output Operations for Pranali, a virtual OS built on top of Linux. Report

ANN for Twitter Sentiment Analysis

Guide : Prof Phuskpak Bhattacharya - Spring 2014

Implemented a Artificial Neural Network to predict sentiments of tweets on twitter. Accuracy achieved was 78%. The feature vector length of size 500 and depth of neural network was 3.

TeamFlowy: Team Management Webapp

Guide : Prof Umesh Bellur - Autumn 2013

Developed webapp to manage teams that supports calendar view of tasks, blogs & reminders. We conceptualized the ER model, normalized the 70+ relations and deployed the system on a Django framework after rigorous testing and optimization using additional indices.

Sequence Alignment on GPUs

Guide : Prof Bernard Menezes - Autumn 2013

Implemented a Sequence Alignment problem on GPUs with parallel version of Needleman- Wunsch algorithm. Investigated Parallel Prefix and Diagonal based approach to solve the prob- lem. Achieved O(n) complexity as compared to qudaratic of the Serial Version. Report

N Body Simulator

Guide : Prof Varsha Apte - Autumn 2012

Designed a simulation showing the interaction between different particles under the effect of intermolecular forces like gravity, electrostatic and nuclear. We used the famous Barnes-Hut Algorithm to optimize computation.
